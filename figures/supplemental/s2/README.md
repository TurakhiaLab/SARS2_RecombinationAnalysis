# Supplemental Figure S2
Comparing recombinant relative fitness in PyR0 vs CoVFit models

The data used to generate both panel plots is in the `static/data` directory.

## Panel a

Run the following script to regenerate the figure S2 plot.
```
python3 plot.py
```


## Panel b

Run the following command to generate figure S2 panel b plot in the localhost browser.
```
python3 app.py panelb.html
```
